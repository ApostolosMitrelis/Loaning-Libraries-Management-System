# 📚 Σύστημα Διαχείρισης Βιβλιοθήκης (Library Management System)

Μια desktop εφαρμογή για τη διαχείριση των βασικών λειτουργιών των δανειστικών βιβλιοθηκών που συμμετέχουν στο δίκτυο, υλοποιημένη με **Python**, **Tkinter** (GUI) και **SQLite**. Το σύστημα παρέχει ξεχωριστά περιβάλλοντα εργασίας για Μέλη και Διαχειριστές (Admin), απλοποιώντας τις διαδικασίες δανεισμού, διαχείρισης προσωπικού και κράτησης χώρων μελέτης.

## 🚀 Λειτουργίες

### 👤 Για Μέλη:
* **Αναζήτηση & Περιήγηση:** Αναζήτηση βιβλίων με βάση τίτλο, συγγραφέα, κατηγορία, βιβλιοθήκη ή γλώσσα.
* **Δανεισμοί & Κρατήσεις:** Προβολή ενεργών δανεισμών, ιστορικού και δυνατότητα κράτησης μη διαθέσιμων αντιτύπων.
* **Κράτηση Χώρων:** Κράτηση αιθουσών μελέτης που προσφέρουν οι βιβλιοθήκες με βάση συγκεκριμένες παροχές (PC, Projector, Κλιματισμός κ.ά.).
* **Διαχείριση Προφίλ:** Προβολή προσωπικών στοιχείων, οφειλόμενων προστίμων και ιστορικού δανεισμών.
* **Αξιολογήσεις:** Προβολή των αξιολογήσεων του χρήστη και δημιουργία νέων για τα βιβλία που έχει δανειστεί.
* **Στατιστικά:** Αναφορές για δημοφιλή βιβλία, βιβλία με την υψηλότερη βαθμολογία και στατιστικά κατηγοριών.

### 🛡️ Για Διαχειριστές:
* **Διαχείριση Υλικού:** Προσθήκη, επεξεργασία και διαγραφή βιβλίων και αντιτύπων.
* **Διαχείριση Χρηστών:** Πλήρης διαχείριση (CRUD) προφίλ Μελών και Προσωπικού.
* **Διαχείριση Βιβλιοθηκών:** Διαχείριση παραρτημάτων βιβλιοθηκών και συνεργαζόμενων μεταφορέων.
* **Διαχείριση Προστίμων:** Παρακολούθηση εκπρόθεσμων δανεισμών και εξόφληση προστίμων.
* **Διαχείριση Δανεισμών:** Καταχώρηση νέων δανεισμών και επιστροφών.

## 🛠️ Τεχνολογίες
* **Γλώσσα:** Python 3.13
* **GUI Framework:** Tkinter
* **Βάση Δεδομένων:** SQLite3
* **Αρχιτεκτονική:** MVC (Model-View-Controller)

## 📂 Δομή Έργου
* `controller.py`: Χειρίζεται τη λογική της εφαρμογής και τη σύνδεση μεταξύ UI και Βάσης. (Controller)
* `view.py`: Περιέχει όλα τα γραφικά στοιχεία (παράθυρα, φόρμες, πίνακες κτλ.). (View)
* `model.py`: Χειρίζεται την επικοινωνία με τη βάση δεδομένων και τα SQL ερωτήματα. (Model)
* `FINAL2.db`: Το αρχείο της βάσης δεδομένων SQLite.

## 🔧 Εγκατάσταση & Εκτέλεση
Για την εγκατάσταση της εφαρμογής μέσω git ακολουθείται η εξής διαδικασία: 
1.  **Κλωνοποίηση του αποθετηρίου:**
* Άνοιγμα ενός παραθύρου Command Line και πλοήγηση στον φάκελο του υπολογιστή που θα θέλατε να γίνει η εγκατάσταση. Έπειτα, επικόλληση της παρακάτω εντολής:
    ```bash
    git clone https://github.com/ApostolosMitrelis/Loaning-Libraries-Management-System.git
    ```
3.  **Μετάβαση στον φάκελο του έργου:**
    ```bash
    cd Loaning-Libraries-Management-System
    ```
5.  **Εκτέλεση της εφαρμογής:**
    ```bash
    python controller.py
    ```
Εναλλακτικά, για την εγκατάσταση και εκτέλεση της εφαρμογής χωρίς την χρήση του git θα πρέπει να γίνει χειροκίνητη εγκατάσταση των απαραίτητων αρχείων στον υπολογιστή. Από αυτό το repository να γίνει εγκατάσταση των αρχείων:
1. controller.py
2. model.py
3. view.py
4. Libraries.db
* Μετά την εγκατάσταση αυτών στον ίδιο φάκελο στον υπολογιστή, με τη χρήση ενός editor ή μέσα από το Command Line να γίνει εκτέλεση του αρχείου controller.py
* Προσοχή: για την επιτυχή εκτέλεση του προγράμματος είναι απαραίτητο να είναι εγκατεστημένες οι εξής βιβλιοθήκες της python: tkinter, datetime και sqlite3 (Όλες συμπεριλαμβάνονται στην Python)
